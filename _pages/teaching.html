  • Computational Methods in Macroeconomics (ECON 402, Fall 2023) [Syllabus]
    Course Description

    This course covers computational methods for solving and analyzing macroeconomic models. Topics include numerical optimization, dynamic programming, simulation techniques, and calibration methods for macroeconomic models.
