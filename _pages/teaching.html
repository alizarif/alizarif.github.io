Instructor

  • Mini-Course on Generative AI Applications [Link]
    Course Description

    This mini-course provides an introduction to generative AI applications with a focus on practical implementation. Students learn about large language models, their capabilities, and how to effectively use them for various applications in economics and data analysis.

  • Computational Macroeconomics
    Course Description

    This course covers computational methods for solving and analyzing macroeconomic models. Topics include numerical optimization, dynamic programming, simulation techniques, and calibration methods for macroeconomic models.

Teaching Assistant

  • Big Data and Machine Learning in Economics
    Course Description

    This course introduces students to big data analysis and machine learning techniques with applications in economics. Topics include predictive modeling, causal inference with machine learning, and analysis of large economic datasets.

  • Intermediate Microeconomics
    Course Description

    This course covers consumer theory, production theory, market structures, and basic game theory. Students learn to apply microeconomic principles to analyze market outcomes and strategic interactions.

  • Introductory Microeconomics and Macroeconomics
    Course Description

    This introductory course covers the fundamental principles of both microeconomics and macroeconomics. Topics include supply and demand, market structures, national income accounting, economic growth, inflation, and monetary policy.
